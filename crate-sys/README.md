# space_engineers_sys

Auto-generated Rust struct and enum definitions from Space Engineers game assemblies.

## Generated Content

| File | Contents |
|------|----------|
| `types.rs` | `MyServerData`, `MyObjectBuilder_World`, replication event parameter types |

## Code Generation

This crate's source files are generated by the C# codegen tool in `../codegen/`.

```powershell
# From repository root
.\codegen\generate.ps1
```

The tool launches Space Engineers, reflects over its assemblies, and outputs Rust definitions with serde + deku derive macros.

## Usage

This crate is an internal dependency of `space_engineers`. Most users should depend on `space_engineers` directly.

```toml
[dependencies]
space_engineers = "1.208015"
```

## Manual Edits

**Do not manually edit `types.rs`** — changes will be overwritten by codegen. If you need to fix a generated type, modify the C# generator instead.
